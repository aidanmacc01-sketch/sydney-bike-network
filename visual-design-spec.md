# 1. Design System Overview

## 1.1 Theme Options

Based on the Stitch frames, the app supports **two themes**:

| Theme | Reference Frame | Use Case |
|-------|-----------------|----------|
| **Dark (Eco-Futurist)** | Top-left, Top-right frames | Night mode, battery saving |
| **Light (Friendly)** | Bottom-left, Bottom-right frames | Default daytime mode |

**Default:** Light theme (Friendly Infrastructure style)

## 2.4 Segment Overlay Rendering

Bike lane segments rendered as **Google Maps Polylines** on top of the base map:

```javascript
// Segment polyline configuration
const segmentPolyline = new google.maps.Polyline({
  path: segmentCoordinates,
  strokeColor: getColorByFacilityType(segment.facility_type),
  strokeOpacity: getOpacityByComfort(segment.comfort_score),
  strokeWeight: 5,
  zIndex: getZIndexByRisk(segment.crash_risk_score)
});
```
